1. Алгоритм — это чёткое и конечное описание последовательности шагов, которые необходимо выполнить для решения поставленной задачи.

2. Алгоритм должен обладать следующими свойствами:
Дискретность — состоит из отдельных шагов.
Определённость — каждый шаг однозначно описан, без двусмысленности.
Конечность — алгоритм завершится за конечное количество шагов.
Результативность — выполнение алгоритма приводит к результату.
Массовость — алгоритм применим к целому классу однотипных задач, а не только к одному конкретному случаю.

3. Линейный алгоритм — это такой алгоритм, где все действия выполняются последовательно, без выбора варианта. Алгоритм с ветвлением включает в себя проверку условий, по результатам которых выбирается один из нескольких путей выполнения.

4. Циклы используются для многократного выполнения набора команд, когда нужно повторить одно и то же действие несколько раз.
  
5. Существуют следующие типы циклов:
Цикл с предусловием (while) — проверка условия происходит перед выполнением тела цикла.
Цикл с постусловием (do while) — проверка условия происходит после выполнения тела цикла.
Цикл-итератор (for) — используется для перебора значений в заданном диапазоне или коллекции.
